# Contributing to OmniSwap SDK

Thank you for your interest in contributing to OmniSwap SDK!

## Development Setup

1. Fork and clone the repository
2. Install dependencies: `npm install`
3. Run tests: `npm test`
4. Build: `npm run build`

## Pull Request Process

1. Create a feature branch from `main`
2. Make your changes with clear, descriptive commits
3. Ensure all tests pass: `npm test`
4. Ensure code compiles: `npm run build`
5. Run linting: `npm run lint`
6. Submit a pull request

## Code Standards

- Follow existing TypeScript patterns
- Add tests for new features
- Consider privacy implications of changes
- Document public APIs

## Reporting Issues

- Use GitHub Issues
- Include reproduction steps
- Specify Node.js and TypeScript versions

## Security

For security vulnerabilities, please email directly rather than opening a public issue.
